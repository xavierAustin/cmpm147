Experiment 02: Living Impressions

Description

Program draws a living impression, or in other words, a generative artwork with moving components based off of an image.

Technical

Using the P5.js library, the program draws a series of lines to a render buffer and a bezier curve slug. On the back end, the slug is a fairly standard Javascript object to allow for movement and deformation.

Reflection

"Every piece of knowledge must have a single, unambiguous, authoritative representation within a system"
